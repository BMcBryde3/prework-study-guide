CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside of an element.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • For-Loops allow you to automate a task on a list/array of variables or values based on the length of the expected number of items
  • Functions allow you block out repeatable steps which can be called throughout your code.
